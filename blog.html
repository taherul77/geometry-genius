DOM is short for "The Document Object model".It is a programming API for HTML and XML documents. It defines a standard for accessing documents and DOM is a platform and language-neutral interface that allows programs and scripts to dynamically access and update the content, structure, and style of a document.

There are 5 ways we can select html elements by using DOM.

  • getElementsByTagName()
  • getElementsByClassName()
  • getElementById()
  • querySelector()
  • querySelectorAll()
    • The greatest usual way to entry an HTML element is to use the id of the element.For example,above the getElementById method used id="demo" to find the element

    Event Bubbling is a concept in the DOM.It happens when an element receives an event,and that event bubbles up to its parent and ancestor elements in the DOM tree until it gets to the root element.So it simply means that the bubbles read-only property of the Event interface indicates whether the event bubbles up through the DOM tree or not.

    A callback function is a function that receives an input from another function and is subsequently called from inside the outer function to finish a task or activity. The aforementioned illustration is a synchronous callback since it is executed right away.When we want one function to execute only after another function has completed its execution, we use callback functions in JavaScript. It needs to pass as a parameter to other functions to make a function callback.
